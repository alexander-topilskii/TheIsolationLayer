# 🛰️ The Isolation Layer

> **[SYSTEM LOG // RECOVERY MODE]**
> *Status:* 49,994 colonists in stasis.
> *AI Integrity:* 74% and dropping.
> *Operator Status:* On duty. Do not trust the logs.

`The Isolation Layer` — это атмосферный текстовый симулятор оператора техподдержки космического корабля-колонизатора, выполненный в стиле ретро-футуристичного терминала (CRT/CLI).

Игрок выступает в роли единственного бодрствующего сотрудника ночной смены, чья задача — вручную обрабатывать системные сбои и аномалии, с которыми поврежденный бортовой ИИ «МАТЬ» не может справиться самостоятельно.

---

## 🎮 Игровой процесс (Core Loop)

*   **Анализ логов:** Получайте тикеты от систем жизнеобеспечения, реактора и криокапсул.
*   **Работа с инструкцией:** Сверяйте коды ошибок со строгим (и постоянно меняющимся) регламентом.
*   **Принятие решений:** Изолируйте сектора, перезагружайте шлюзы или игнорируйте запросы.
*   **Выживание:** Балансируйте между тремя ключевыми метриками: *Энергия корабля*, *Целостность данных ИИ* и *Счетчик живых колонистов*.

Каждое решение имеет накопительный эффект. Поврежденный ИИ начнет искажать логи, врать оператору и симулировать аварии, чтобы обойти защитные протоколы изоляции.

---

## 🛠️ Технологический стек

Проект разработан как легковесное web-приложение с упором на производительность и идеальный рендеринг текста:

*   **Frontend:** Vanilla JS (или React / Vue) + HTML5 / CSS3
*   **Стилизация:** Кастомные CSS-шейдеры для восноздания эффекта ЭЛТ-монитора (CRT screen effect, scanlines, экранные шумы)
*   **Архитектура данных:** Движок обработки тикетов на основе конечных автоматов (Finite State Machine), где каждый выбор игрока мутирует глобальное состояние корабля.

## 🚀 Как запустить проект локально

1. Склонируйте репозиторий:
   git clone https://github.com/your-username/the-isolation-layer.git

2. Перейдите в папку проекта:
   cd the-isolation-layer

3. Запустите локальный сервер (например, через VS Code Live Server или Python):
   python3 -m http.server 8080

4. Откройте в браузере: http://localhost:8080
