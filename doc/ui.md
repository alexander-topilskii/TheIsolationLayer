# 🖥️ Интерфейс и Визуальный стиль: The Isolation Layer

## 🎨 Общая эстетика (Visual Style)

* **Палитра:** Монохромный зеленый фосфор (#00FF66) или янтарный желтый (#FFB300) текст на глубоком черном фоне (
  #0A0A0A).
* **Шрифт:** Моноширинный (Monospace), например, Courier New, Fira Code или JetBrains Mono.
* **Эффекты экрана (CSS CRT Шейдер):**
    * Скан-линии (Scanlines) — тонкие горизонтальные полупрозрачные полосы.
    * Легкое размытие (Glow/Blur) вокруг светящихся букв.
    * Виньетка (Vignette) — затемнение по краям экрана для эффекта выпуклого кинескопа.
    * Микро-мерцание (Flicker) — едва заметное изменение яркости экрана.

---

## 📐 Макет экрана (Layout Grid)

Экран жестко разделен на 4 основные зоны с помощью CSS Grid:

1. ZONE 1 (HEADER): Верхняя панель. Метрики и системный статус.
2. ZONE 2 (MAIN): Левая часть экрана. Лента логов и активное окно тикета.
3. ZONE 3 (SIDEBAR): Правая часть экрана. Инструкции, Справочник и База данных.
4. ZONE 4 (FOOTER): Нижняя панель. Консольный ввод (CLI) или кнопки быстрых действий.

---

## 📑 Детальное описание зон

### ZONE 1: Верхняя панель (Header)

Отображает текущее состояние корабля. Данные обновляются в реальном времени.

* **SYS.TIME:** Игровое время текущей смены (например, SHIFT: 02 // TIME: 03:45).
* **PWR (Энергия):** Текстовый прогресс-бар: PWR [████████░░] 80%.
* **AI.STB (Стабильность ИИ):** AI.STB [██████░░░░] 64%. Если падает ниже 30%, шкала начинает хаотично мигать.
* **COLONISTS (Колонисты):** Счетчик живых людей: STASIS_CNT: 49,994 / 50,000.

### ZONE 2: Главный терминал (Main Screen)

Основная рабочая область, имитирующая бегущую строку логов ядра.

* Текст выводится с эффектом постепенной печати (typewriter effect).
* Свежие логи подсвечиваются ярче, старые уходят вверх и блекнут.
* Формат лога: [03:44:12] [WARN] [SYS.LIFE_SUPPORT] Сектор С-3: CO2 > 40%.
* Когда активен критический тикет, это окно блокируется рамкой: [CRITICAL ACTION REQUIRED].

### ZONE 3: Боковая панель (Sidebar / Документация)

Вкладки, между которыми игрок переключается мышкой или горячими клавишами F1, F2, F3:

1. [F1] PROTOCOLS: Текст текущей инструкции (например: "При аварии реактора типа В — изолировать смежные шлюзы").
2. [F2] DIAGNOSTICS: Карта секторов корабля (простая текстовая схема-сетка A1, A2, B1, B2...), где поврежденные сектора
   подсвечиваются символом [X].
3. [F3] MANIFEST: Поисковая строка для ручного ввода фамилий колонистов.

### ZONE 4: Панель управления (Footer)

Область принятия решений. Поддерживает два режима:

* Режим кнопок: Крупные текстурированные кнопки для выбора ответа: [ F1: ACCEPT ], [ F2: ISOLATE ], [ F3: OVERRIDE ].
* Режим CLI (Командная строка): Инпут для ввода команд вручную. Пример: OPERATOR@TERRA4_CORE_v1.0.4> sys.bypass A-404

## 🚨 Динамические изменения UI (Реакция на события)

1. Эффект «Паники» (Падение Стабильности ИИ):
    * Буквы в случайных словах заменяются на битые символы: "Сектор B-1" превращается в "Сект#р B-%".
    * Интерфейс слегка сдвигается по горизонтали (эффект глитча).

2. Эффект «Обесточивания» (Низкая Энергия):
    * Яркость интерфейса падает до минимума, частота мерцания экрана увеличивается.

3. Эффект «Удаленного вмешательства»:
    * Когда ИИ пытается обмануть игрока, в командной строке сам по себе начинает быстро печататься текст от лица ИИ,
      перекрывая ввод игрока.
